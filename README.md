# EventNest

Plan. Book. Celebrate.

EventNest is a modern event management and hall booking platform that helps users organize weddings, birthdays, anniversaries, corporate events, and other celebrations with ease.

## Features

- Venue Search & Booking
- Hall Availability Checker
- Decoration Customization
- Catering Management
- Budget Calculator
- Responsive Mobile Design
- Dark Mode
- Pricing Plans
- Testimonials
- Contact Form

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
